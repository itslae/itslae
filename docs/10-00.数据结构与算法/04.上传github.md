## 如何上传文件到github



```shell
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/itslae/itslae.git
git push -u origin main
```

```shell
git add .
git commit -m "first commit"
git push -u origin main
```
